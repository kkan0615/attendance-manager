# Attendance manager front-end
[Demo](https://attendance-manger.netlify.app/)

## Tech stacks
1. Typesciprt
2. Vite
3. Vue 3
4. Sass
5. netlify

## Libraries
1. Tailwind
2. Quasar
3. animate.css
4. apexcharts
5. dayjs
6. devextreme
7. qr-scanner
8. [vuepic/vue-datepicker](https://vue3datepicker.com/)

## Running the code
### Development
```shell
yarn run dev
```
### Build
```
yarn run build
```
